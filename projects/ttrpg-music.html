TTRPG Music Manager

Tech Stack: React, Flask, Redux

Github


This was a personal project I created to manage playing different emotions of music for my players in TTRPGs. I created it with a few different requirements in mind:

  1. The app needs to be able to add MP3s and WAVs to different playlists
  2. The app needs to be able to add and edit playlists
  3. The app needs to fade out the currently playing playlist to play another when another is selected
  4. The app needs an ambient sound section that can play in the background of any other song
  5. The app needs to be able to select and play specific songs, when necessary

Some key systems I developed:

  • Using Material UI to create expandable lists as well as other dropdown lists
  • A Redux state manager that controls the "group" being played at any given time
  • Simple, custom colored UI that is intuitive for any user
