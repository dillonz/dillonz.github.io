Legend of Zelda NES - Remake

Tech Stack: Unity, C#

Link to Windows Build: Legend of Zelda NES - Remake

This recreation of the first dungeon from the original Legend of Zelda was created over the course of about a month in a randomly assigned group of three. It was the first major game development I did in Unity, and we ran into some hiccups with a non-communicative, procrastinating team member. On many occasions, the other teammate and I would have to take on rewriting some of their code to make the game work as expected.

We were also tasked with introducing a new mechanic to the game that wasn't in it before, so we created an extra level (accessible by clicking on the main menu) that involves more modern Zelda block puzzles.

Controls:

  • WASD - Move
  • . - Use Sword
  • / - Use item
  • RShift - Change Item

Some key contributions by me:

  • All enemy AIs except for the boss
  • Spawning, despawning, and remembering all enemies and where they are/whether or not they've been killed before
  • Weapon switching and usage
  • Most, if not all, animations of enemies and the player
  • Player movement with grid snapping
