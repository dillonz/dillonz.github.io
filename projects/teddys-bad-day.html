Teddy's Bad Day

Tech Stack: Unity, C#

Link to Windows Build: Teddy's Bad Day

This was created for the GMTK Game Jam in 2021 with the theme of Joined Together.

A big reason I did this was to teach my brother game development with Unity, so much of my time was spent teaching and training him up on game development paradigms and how they are different/similar to typical dev paradigms.

We created this game over the course of about 15-20 hours each, so it wasn't entirely complete, but we got the main idea of what we wanted to do across!
